## Deploying to Vercel ğŸš€

Vercel is the easiest way to deploy your Next.js app - **completely free for personal projects**!

### Quick Deploy Steps

1. **Push to GitHub**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin <your-github-repo-url>
   git push -u origin main
   ```

2. **Deploy with Vercel**:
   - Go to [vercel.com](https://vercel.com/new)
   - Sign up or log in with your GitHub account
   - Click "Add New Project"
   - Import your GitHub repository
   - Click "Deploy" (Vercel auto-detects Next.js settings!)

3. **Done!** Your site is live at `https://your-project.vercel.app`

### Automatic Deployments

Every push to your main branch will automatically deploy updates to your live site. Pull requests create preview deployments for testing.

Check out the [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying) for more details.
